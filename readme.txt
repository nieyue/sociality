该系统的后台管理一共有3个功能模块
1.管理用户：添加、删除用户。
2.管理用户朋友圈：对用户发布的违规朋友圈进行删除。
3.发布活动、公告：定期发布活动公告组织用户参加。
该系统的前台管理一共有5个功能模块
1.用户自身信息修改功能：上传头像，填写自身信息。
2.发时光圈功能：用户可以发朋友圈到自己的页面，可以在此页面与他人互动，可以给他人点赞并关注他人为好友。
3.添加/删除好友功能：通过账号来添加删除好友。
4.聊天功能：可以与好友聊天。
5.创建聊天房间功能：通过创建自己喜欢的语音房或者电影房来邀请其他用户加入在看电影听歌的同时聊天。
对于用户来说，能在前台进行个人信息修改、添加照片、添加好友、用户相互聊天、创建聊天房、动态交互，对于后台管理员来说可以查看用户的信息将不当信息进行删除。
要求有后台数据库
